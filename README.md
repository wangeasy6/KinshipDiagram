# Kinship Diagram
编织一张自己的**亲戚图谱**，让回家过年的你不再因为不认识人而社恐。

<br/>

## Development Environment

* Qt Creator 15.0.1
* Kit: Qt 6.6.3 MinGW 64-bit
* Compiler: MinGW 11.2.0 64-bit
* OS: Windows 10 Version 22H2
* Architecture: x86_64

<br/>
